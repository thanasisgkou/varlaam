Διαχείριση
Πώς δουλεύει το Admin Panel

Το admin panel είναι στο /admin και τρέχει με Decap CMS (πρώην Netlify CMS). Χρησιμοποιεί Git Gateway backend — δηλαδή κάθε αλλαγή γίνεται commit στο Git repo και trigger αυτόματο rebuild στο Netlify.

📰 Νέα & Ανακοινώσεις

  • Τίτλος, ημερομηνία, περίληψη
  • Εικόνα (upload μέσω CMS)
  • Συντάκτης (default: Σύλλογος Βαρλαάμ)
  • Κείμενο σε Markdown editor
  • Αποθηκεύεται σε src/content/news/

🎉 Εκδηλώσεις

  • Τίτλος, ημερομηνία έναρξης/λήξης
  • Τοποθεσία, περίληψη
  • Εικόνα (optional)
  • Κείμενο σε Markdown
  • Αποθηκεύεται σε src/content/events/

🖼️ Γκαλερί

  • Τίτλος, κατηγορία (dropdown)
  • Κατηγορίες: Χωριό, Εκδηλώσεις, Παλιές Φωτ., Φύση
  • Upload εικόνας
  • Περιγραφή (optional)
  • Αποθηκεύεται σε src/content/gallery/ (JSON)

🌍 Απόδημοι

  • Ιστορίες: Όνομα, τοποθεσία, περίληψη, κείμενο
  • Πίνακας Ανακοινώσεων: Τίτλος, συντάκτης
  • Πεδίο «Εγκεκριμένο» (boolean) για moderation
  • Upload σε diaspora-stories/ & diaspora-bulletin/

🔐 Πρόσβαση στο Admin

1. Πηγαίνεις στο varlaam.netlify.app/admin
2. Κάνεις login μέσω GitHub (Git Gateway)
3. Βλέπεις τις 5 collections (Νέα, Εκδηλώσεις, Γκαλερί, Ιστορίες, Ανακοινώσεις)
4. Δημιουργείς/επεξεργάζεσαι content με visual editor
5. Πατάς "Publish" → γίνεται Git commit → auto rebuild → live σε ~1 λεπτό

Σημείωση: Πρέπει να ενεργοποιηθεί το Git Gateway στο Netlify dashboard (Settings → Identity → Git Gateway).
